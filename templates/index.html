Quantum Random Number Generator

• Click "Generate" to get a random number.

This website was designed using HTML and CSS and implemented using Flask. The purpose of this website is to demostrate the property of qubits by using them to generate random numbers. The random numbers are any 16 bit number, since it was implemented using 16 qubits. The way it works is by creating a quantum circuit with 16 Qubits and 16 Classical bits which they map to. The qubits then have a Hadamard gate applied to all of them in order to put them in a superposition state. Then, I simply measure the qubits and store that value within the classical qubits. This was strictly a educational project, and one to help develop my skills in Qiskit, Quantum Computing Concepts, and web development. The circuit can be seen below.
